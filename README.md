Employee Management System (EMS)

Main Features
Add a new employee record (Insertion at tail)
Display all active employee records (List traversal)
Search an employee by unique ID (Linear search optimization)
Delete an employee record by ID (Node deletion and pointer realignment)
Real-time Dynamic Table View (GUI grid presentation)
Input Validation & Error Alerts (Ensuring correct data format for ID and Salary)

How to Run the Project
Install a C++ compiler (such as MinGW/G++) and Python 3.x environment.
Compile the C++ core engine: g++ ems_engine.cpp -o ems_engine
Ensure the compiled binary is in the same directory as the GUI script.
Run the graphical interface application: python ems_gui.py
Use the modern interactive GUI window to manage employee records seamlessly.

Project Description
This Employee Management System is a hybrid Data Structures project that combines the high performance of C++ with a modern Python Graphical User Interface (GUI). The backend uses a Singly Linked List to manage dynamic records in memory heap efficiently, avoiding structural overflows. The frontend eliminates standard CLI black screens, providing HR managers with interactive input forms, safe field validation, dynamic grid updating, and direct binary communication for real-time CRUD operations.
